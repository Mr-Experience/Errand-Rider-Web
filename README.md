# Errand Riders 🚚

Errand Riders is a comprehensive delivery service platform for food, local markets, and supermarkets.

## About This Repository

This repository contains the web-based applications for the Errand Riders ecosystem, which includes:
- **Main Website**: The landing page and public-facing site.
- **Admin Dashboard**: The centralized control panel for managing operations, users, and platform settings.
- **Riders Dashboard**: The portal for delivery personnel to manage their profiles, earnings, and operations.
- **Vendors Dashboard**: The portal for merchants (restaurants, supermarkets, and local markets) to manage their store, inventory, and orders.

> **Note on the User App:** The main customer-facing mobile application used by users to place orders is maintained in a separate codebase (`Errand Riders Mobile`).

## Development

This project was bootstrapped with React and Vite.

### Running the Project

```bash
npm install
npm run dev
```
